# Hacknet Methodology

This repository is a guide for working through the Hacknet environment.

It's mainly built to help beginners get started and gain hands-on experience with hacking. It’s not meant to hand you all the answers, but to support your learning. Hacknet can feel overwhelming at first, and it’s not always obvious where to begin.

<br>

> [!CAUTION]
> ## Disclaimer
> All testing must remain within the authorized IP range assigned to this environment. Any activity conducted outside of the approved scope is considered a violation of the rules and may result in immediate and permanent removal from Hacknet.
